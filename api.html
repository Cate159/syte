📚 API Pubbliche

Chiunque può usare queste API per creare app, bot o integrazioni!

GET Statistiche Globali

/api/public/stats

Restituisce statistiche globali del gioco

GET Top Giocatori

/api/public/top/:limit

Ottieni i migliori N giocatori (max 100)

GET Info Giocatore

/api/public/player/:name

Ottieni informazioni su un giocatore specifico

GET Consiglio Casuale

/api/public/random-tip

Un consiglio casuale per il gioco

GET Contatore Visite

/api/visits

Numero totale di visite al sito

GET Classifica Completa

/api/leaderboard

Classifica completa dei migliori 100 giocatori

💻 Esempio di utilizzo

// JavaScript fetch('https://tuo-sito.onrender.com/api/public/stats') .then(r => r.json()) .then(data => console.log(data)); // Output: { total_visits: 150, total_players: 23, total_score: 50000 }
